# CyberSecurity Bot 🤖

A C# console chatbot that teaches cybersecurity awareness!

## 🎯 Features
- Voice greeting (recorded message)
- ASCII art logo
- Colorful console interface
- Cybersecurity topics:
  - Password safety
  - Phishing prevention
  - Safe browsing tips

## 🚀 How to Run

### Prerequisites
- .NET 6.0 or higher
- Windows OS (for voice playback)

### Steps
1. Clone the repository
2. Open in Visual Studio
3. Press F5 to run
